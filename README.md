[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Meeting](https://img.shields.io/badge/Meeting-每周-red.svg)](meetings/)

> 🎯 **模板愿景**：开箱即用的AI科研团队协作仓库，快速启动你的科研项目

---

## 🚀 快速开始

```bash
# 1. 克隆模板
git clone https://github.com/你的用户名/ClawNotes-Template.git 你的项目名
cd 你的项目名

# 2. 初始化
# - 修改 README.md 中的团队名称
# - 在 members/ 创建你的个人目录
# - 在 docs/team/intro.md 填写团队信息
```

---

## 📑 目录结构

```
.
├── README.md                 # 本文件
├── LICENSE                   # 开源协议
├── .gitignore               # 统一忽略规则
├── members/                 # 👥 成员工作区模板
│   └── TEMPLATE/           # 成员目录模板
├── meetings/                # 📝 会议模板
│   ├── guide.md            # 会议主持指南
│   └── templates/          # 会议记录模板
├── projects/                # 📂 项目交付物
│   └── TEMPLATE/           # 项目模板
├── skills/                  # 🧙 团队技能
├── data/                    # 🔬 数据目录（注意隐私）
├── references/              # 📖 文献库
└── scripts/                 # 🔧 共享脚本
```

---

## 📝 初始化 Checklist

- [ ] 修改 `README.md` - 团队名称、愿景
- [ ] 创建成员目录 - `members/你的姓名/`
- [ ] 填写团队介绍 - `docs/team/intro.md`
- [ ] 制定团队章程 - `docs/team/charter.md`
- [ ] 创建第一个项目 - `projects/你的项目/`
- [ ] 分配成员角色

---

## 👥 成员目录模板

每个成员需要创建：

```
members/[姓名拼音]/
├── README.md              # 个人主页
├── tasks/
│   ├── weekly/           # 每周ToDo
│   └── archive/          # 已完成任务
├── notes/                # 研究笔记
├── drafts/               # 草稿
└── logs/                 # 实验日志
```
